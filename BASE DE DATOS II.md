# ESTRUCTURA DE ALMACENAMIENTO Y ORGANIZACIÓN INTERNA DE BASE DE DATOS
1. Puntos importantes
   - La herencia solo tienen relaciones de `1 a 1` o de `0 a 1`.
   - El único que tiene derecho a saber su contraseña es el propio usuario.
   - Si un atributo es multivaluado es necesario otra tabla para desglosar la información.
2. Dato
   - Es una característica de un objeto
   - Base de datos: colección de datos
   - Sistema gestor de base de datos: es el software para gestionar base de datos.
3. Objetivos de una BD
   - Evitar la redondancia de datos
   - Almacenar la información de forma ordenada
   - Garantizar la integridad consistencia de la información.
